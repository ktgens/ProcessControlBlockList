# ProcessControlBlockList



Программа представляет собой реализацию класса, для управления связанным списком блоков управления процессами (PCB).



PCB (Process Control Block) представлен в виде структуры с полями:

\- processID: Идентификатор процесса.

\- processName: Название процесса.

\- processStatus: Статус процесса (Running, Waiting, Stopped).

\- commandCounter: Счетчик команд.

\- сpuRegisters: Массив для хранения состояния регистров CPU.



Класс ProcessList представляет собой реализацию односвязного списка, где элементы не могут повторяться и добавляются по возрастанию processID. Реализованы операции вставки, вывода, удаления элемента по ID.
